Hello, to run the program: 
- In the user's IDE of choice clone this repository (or open using VS codespace)
- Navigate to a terminal within the IDE
- Change the directory to the project folder where the repo is saved locally
- The program __requires the python module 'questionary'__ to be installed. This can be done using the command: 'pip install questionary' (make sure pip is installed). Note you can also use a package manager such as conda-forge and use the command: 'conda install conda-forge::questionary'.
- Now run the program by typing __'python main.py'__. If this does not work try typing 'python3 main.py'.
- A UI will now appear asking the user to use the arrow keys to select the column to sort. With the column highlighted, press the enter key to begin sorting using merge sort and quick sort.
- The time it takes the program to execute depends on the column selected to sort. Please refer to our tested execution times below and note these may differ for your machine.

* Year
    - Merge Sort: <1 sec
    - Quick Sort: 252 sec
* Gender
    - Merge Sort: <1 sec
    - Quick Sort: 232 sec
* Age
    - Merge Sort: <1 sec
    - Quick Sort: 5 sec
* Location
    - Merge Sort: <1 sec
    - Quick Sort: 8 sec
* Smoking history
    - Merge Sort: <1 sec
    - Quick Sort: 124 sec
* bmi
    - Merge Sort: <1 sec
    - Quick Sort: 25 sec
* hbA1c
    - Merge Sort: <1 sec
    - Quick Sort: 28 sec
* Glucose level
